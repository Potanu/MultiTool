# MultiTool
シンプルなお買い物メモアプリです。  
チェックリスト一覧ページと、購入ログ一覧ページの２画面で構成されています。

## アプリ概要
1. 「追加」ボタン押下で項目追加
2. チェックボックスにチェックを付けると出現する「購入済みにする」ボタンを押下する
3. 2でチェックを付けた商品が購入ログカレンダーページの該当の日付に表示される
